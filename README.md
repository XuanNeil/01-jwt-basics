### jsonwebtoken basics
## Getting Started

1. First, install node modules:

```bash
npm install
```

2. Second, create .env file 

```bash
Example:
## Environment ##
NODE_ENV= 

## Server ##
PORT=
HOST=localhost
ACCESS_TOKEN_KEY=
REFRESH_TOKEN_KEY=
ACCESS_TOKEN_LIFE=
REFRESH_TOKEN_LIFE=

```

3. After, run the development server:

```bash
npm run dev
```

Open http://localhost:{port} with your browser to see the result.
